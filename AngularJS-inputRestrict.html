AngularJS Directive for restricting the input values typed in to an input element based on a REGEX pattern.
Digits : Model Value: {{val1}}
Alpanumerics or underscore : Model Value: {{val2}}
Lower-Case Letters : Model Value: {{val3}}
Upper-Case Letters : Model Value: {{val4}}
Percentage value(0 to 100) with upto two decimal places : Model Value: {{val5}}
